Installation
01
Clone or extract the template

Download the zip from your purchase receipt and extract it, or clone from your delivery repo.

bash 
# Extract the downloaded zip
unzip devnoxxkit-v2.1.zip -d my-project
cd my-project
02
Open the default homepage

This is a static HTML package. Open index.html in your browser, or open any page directly.

03
Optional: run a local static server

For best results (fonts, relative paths), serve the folder with any static server.

✓ Tip: This is optional. The pages also work when opened directly.

Changing the accent color

The entire theme is driven by --gold. To create a teal agency theme, simply update this single token:

CSS
/* In each HTML page :root */
:root {
  /* Default: Cinematic Gold */
  --gold: #C9A96E;
  --gold-light: #DFC28E;
  --gold-dim: rgba(201, 169, 110, 0.12);
  --border-gold: rgba(201, 169, 110, 0.3);

  /* Swap to: Electric Blue */
  /* --gold: #4B8CF5; */
  /* --gold-light: #7AAFF8; */
  /* --gold-dim: rgba(75, 140, 245, 0.12); */
}
Typography

Three font families form the DevnoxxKit type system. All loaded from Google Fonts with display=swap for performance.

VariableFontUsageWeights
--font-displayFraunces (Variable)Headings, hero titles, pull quotes100–900 + Italic
--font-bodyDM Sans (Variable)Body text, UI labels, navigation300–700
--font-monoJetBrains MonoLabels, tags, code, metadata400, 500
Type scale classes
CSS
/* Fluid sizing: min / preferred / max */
.display-xl  { font-size: clamp(64px, 10vw, 148px); }  /* Hero */
.display-lg  { font-size: clamp(44px, 6vw, 96px); }   /* Section hero */
.display-md  { font-size: clamp(28px, 3.5vw, 52px); } /* Section title */
.heading-sm  { font-size: clamp(20px, 2vw, 30px); }  /* Card headings */
.label       { font-family: var(--font-mono); font-size: 11px; letter-spacing: 0.12em; }
Changing fonts

Replace the Google Fonts import URL and the three CSS variables. The entire template updates automatically.

HTML
<!-- Replace in the <head> of every page -->
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@400;700&display=swap" rel="stylesheet"/>

/* Then update in :root */
--font-display: 'YourFont', Georgia, serif;
--font-body: 'YourBodyFont', system-ui, sans-serif;

Customization Guide

DevnoxxKit is engineered to be customized in minutes, not hours. Here's the fast path to making it fully yours.

1. One-file brand change

Open any HTML page and edit the :root block at the top. Change the core variables and the entire site repaints.

2. Replace placeholder content

Search for DevnoxxKit, chtauqeer1117@gmail.com, and Everywhere + Nowhere across all HTML files. Replace with client content.

3. Replace Unsplash images

All images use Unsplash source URLs. Replace https://images.unsplash.com/ references with your own assets in public/assets/.

4. Connect the contact form
HTML
<!-- Replace YOUR_ID with your Formspree endpoint ID -->
<form action="https://formspree.io/f/YOUR_ID" method="POST">

<!-- Or for Netlify Forms, replace with: -->
<form name="contact" method="POST" data-netlify="true">

SEO Guide

DevnoxxKit ships with a clean semantic HTML structure. Here's the minimum SEO setup required for every page:

HTML
<head>
  <!-- Required for every page -->
  <title>Page Title — Site Name</title>
  <meta name="description" content="150–160 chars. Unique per page."/>
  <link rel="canonical" href="https://yourdomain.com/this-page"/>

  <!-- Open Graph (social sharing) -->
  <meta property="og:title" content="Page Title"/>
  <meta property="og:description" content="Same as meta description."/>
  <meta property="og:image" content="https://yourdomain.com/og-image.jpg"/>
  <meta property="og:type" content="website"/>

  <!-- Structured data (optional but recommended) -->
  <script type="application/ld+json">
  { "@context": "https://schema.org", "@type": "Organization",
    "name": "Your Agency", "url": "https://yourdomain.com" }
  </script>
</head>
