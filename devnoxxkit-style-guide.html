Alerts & Callouts

Four semantic variants. Always include a descriptive icon. Use role="alert" for errors, role="status" for informational.

Error: Something went wrong. Please check your connection and try again.

Dark / Light Mode Toggle

System preference detection + localStorage persistence. Click the toggle below — your preference is saved.

Click to toggle — preference saved to localStorage
Dark/Light toggle — functional demo with system preference detection
JavaScript · Dark/Light persistence
// Detects system preference, reads localStorage, applies theme const STORAGE_KEY = 'dnk-theme'; const root = document.documentElement; // On load — apply saved or system preference const saved = localStorage.getItem(STORAGE_KEY); const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches; const theme = saved ?? (prefersDark ? 'dark' : 'light'); root.setAttribute('data-theme', theme); // Toggle function function toggleTheme() { const next = root.getAttribute('data-theme') === 'dark' ? 'light' : 'dark'; root.setAttribute('data-theme', next); localStorage.setItem(STORAGE_KEY, next); }
14 — Patterns

Motion Tokens

Three easing curves and three duration tiers. Use the same motion language throughout — never mix different easing styles on adjacent elements.

Easing Curves
--ease (Spring Out)
cubic-bezier(0.16, 1, 0.3, 1)
Default for all reveals, hover states
--ease-in-out (Smooth)
cubic-bezier(0.65, 0, 0.35, 1)
For modals, drawers, heavy transitions
ease-in (Enter)
cubic-bezier(0.4, 0, 1, 1)
For exit animations only
Duration Tiers
Fast · 150–200ms
Hover states, color changes, opacity
Default · 300–500ms
Transform, border, background transitions
Slow · 700–1000ms
Scroll reveals, page load animations
